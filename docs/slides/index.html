What's an Angular 2 Directive?

Combination of Directive and Controller from Angular 1.x

Types of Directives

  1. Structural: ngIf, ngFor, ngSwitch
  2. Attribute: disabled, ngModel
  3. Component

Syntax


@Component({
    selector: 'mySelector',
    templateUrl: './myTemplate.html',
    directives: [ChildComponent1, ChildComponent2],
    pipes: [UsedPipe1],
    styles: ['background-color: #fff'],
    providers: [UsedService1, UsedService2]
})

TypeSyntax
Element Namename
