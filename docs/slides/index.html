What's an Angular 2 Directive?

Combination of Directive and Controller from Angular 1.x

Types of Directives

  1. Structural: ngIf, ngFor, ngSwitch
  2. Attribute: disabled, ngModel
  3. Component

Syntax


@Component({
    selector: 'mySelector',
    templateUrl: './myTemplate.html',
    directives: [ChildComponent1, ChildComponent2],
    pipes: [UsedPipe1],
    styles: ['background-color: #fff'],
    providers: [UsedService1, UsedService2]
})

Selector

  1. Element Name: 'name'
  2. Attribute: '[name]'
  3. Attribute & Value: '[name=value]'
  4. CSS Class: '.name'
  5. Negation: ':not(name)'
  6. Multiple (any match): 'selector1, selector2'

Dependency Injection

  • Providers (Services)

  • Direct child elements
